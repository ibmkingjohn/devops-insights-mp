-- SUMMARY --

This is a views display specifically for embed use. It automatically allows
embedded views to use exposed filters and allows filters to be placed in a
block or embedded separately from the view.

-- REQUIREMENTS --

* Views 7.x-3.x

-- SUGGESTED MODULES --

* Advanced help 7.x-1.x

-- INSTALLATION --

* Install as usual, see (http://drupal.org/documentation/install/modules-themes/modules-7)
  for further information.

* To add embed views to the View UI navigate to
  "Administration >> Structure >> Views >> Settings" (admin/structure/views/settings)
  and check the "Show the embed display in the ui" check box.
